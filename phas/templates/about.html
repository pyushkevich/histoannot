About PICSL Histology Annotation Server

PHAS (PICSL Histology Annotation Server) is a web-based platform for viewing, annotating, and analyzing large-scale histology slide images.

Key Features

  • High-Resolution Viewing: Seamlessly navigate whole-slide images using OpenSeadragon's deep zoom technology
  • Annotation Tools: Create and manage annotations on histology slides
  • Deep Learning Integration: Support for segmentation and classification workflows
  • Project Management: Organize slides into projects with user access control
  • Cloud Storage: Integration with Google Cloud Storage for scalable data management

Technology

PHAS is built with Python Flask and leverages OpenSlide for efficient processing of whole-slide imaging formats. The platform supports distributed processing and collaborative workflows for histology research.
